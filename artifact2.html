Introduction

The AI-Powered Sentiment Analysis Dashboard is an intelligent system that uses Natural Language Processing (NLP) and Machine Learning (ML) to analyze textual data and classify sentiments as positive, negative, or neutral. This project demonstrates Krishnapraneeth’s ability to combine data preprocessing, model training, evaluation, and visualization into a cohesive AI workflow.

Objective

To design and implement an ML-based sentiment analysis model that enables businesses and analysts to visualize customer opinions, track satisfaction trends, and make data-driven decisions.

Process

  1. Data Collection: Used Kaggle product reviews dataset containing labeled sentiments.
  2. Data Preprocessing: Cleaned text data by tokenization, stopword removal, and lemmatization.
  3. Feature Engineering: Applied TF-IDF vectorization to convert text into numerical features.
  4. Model Training: Trained and tested Logistic Regression, Naïve Bayes, and SVM models.
  5. Evaluation: Compared accuracy and F1-score to determine the best-performing model.
  6. Visualization: Created interactive sentiment plots using Matplotlib and Plotly.
  7. Deployment: Hosted results and reports on GitHub Pages.

Tools & Technologies

  • Languages: Python
  • Libraries: Pandas, NumPy, Scikit-learn, Matplotlib, Plotly, NLTK
  • Environment: Jupyter Notebook
  • Hosting: GitHub Pages
